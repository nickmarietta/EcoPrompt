# EcoPrompt

EcoPrompt is a prompt optimization tool: paste a verbose prompt and get a shorter, clearer version with token savings and an efficiency readout. Built with [Next.js](https://nextjs.org) (App Router), React, and Tailwind CSS.

## How to run this project

Follow these steps in order:

1. **Install Node.js**  
   Use a current LTS version (18+ recommended).  

2. **Install dependencies** (from the project root):

   ```bash
   npm install
   ```

3. **Start the development server**:

   ```bash
   npm run dev
   ```

4. **Open the app** in your browser:

   **http://localhost:3000**

5. **Stop the server** when you’re done: press `Ctrl+C` in the terminal.
